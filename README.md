# Calculadora Web

Proyecto colaborativo de **Despliegue de Aplicaciones Web**.  
La aplicación es una calculadora básica con interfaz web que permite introducir dos números y realizar operaciones de **suma, resta, multiplicación y división**, mostrando el resultado en pantalla.

---

## Equipo de trabajo

- **Integrante A**: responsable de crear el repositorio, la estructura inicial y las ramas de trabajo. Implementa suma y resta.
- **Integrante B**: responsable de la interfaz y de las funcionalidades de multiplicación y división.
- Ambos trabajan con el mismo usuario de GitHub (`VicenteMauro`) en directorios locales separados (`calculadora-web_A` y `calculadora-web_B`).
